Adminish

A delightful way to describe your REST API

Example Usage

                
                  # app/controllers/admin_controller.rb

                    def models
                    # the array is an array of activerecord classes you want to have managed for you
                      render json: Adminish::Admin.new([Post, Category])
                    end
